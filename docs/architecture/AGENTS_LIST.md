## 如何使用

### 在 Claude Code 中使用 Agents

1. **查看可用 agents**
   ```bash
   ls ~/.claude/agents/
   ```

2. **快速查找特定 agent**
   ```bash
   ls ~/.claude/agents/ | grep -i "keyword"
   ```

3. **使用 Agent Tool**
   ```
   在 Claude Code 中使用 @agent 或通过 Agent 工具调用
   ```

### 更新 Agents

```bash
# 转换所有新增的 agents
./scripts/convert.sh

# 安装到 Claude Code
./scripts/install.sh --tool claude-code

# 验证 agents
./scripts/lint-agents.sh
```
